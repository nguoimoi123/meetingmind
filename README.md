## Giới thiệu

Đây là dự án trợ lý cuộc họp thông minh, được xây dựng dưới dạng một hệ thống nhiều phần gồm ứng dụng Flutter, backend Flask và dashboard admin riêng bằng React + Vite.

MeetingMind hỗ trợ ghi âm, chép lời realtime, tóm tắt bằng AI, tạo action items, quản lý notebook, nhắc việc và cộng tác nhóm.

## Tính năng chính

- Đăng nhập bằng Google / Email
- Ghi âm và chép lời cuộc họp realtime
- Tóm tắt nội dung cuộc họp bằng AI
- Tạo action items và key decisions
- Chat với nội dung cuộc họp / notebook
- Quản lý notebook và tri thức liên quan
- Nhắc việc và quản lý lịch
- Dashboard admin riêng cho quản trị

## Cấu trúc ứng dụng

- `meetingmind_ai/`: ứng dụng Flutter chính
- `meetingmind_ai_backend/`: backend Flask và realtime services
- `meetingmind_admin_web/`: dashboard admin bằng React + Vite
